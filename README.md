# Detect-Face-IronMan
 ### 1. 프로젝트명
           * openCV와 학습 데이터를 사용한 얼굴 감지 마스킹 프로그램
  
 ### 2. 프로그램 설명 
           * 학습된 XML 데이터를 이용해 웹 캠의 얼굴점을 인식, 지정 이미지를 웹 캠의 화면과 합성
           * 화면에 비춰지는 얼굴 크기에 따라 지정 이미지의 크기 또한 조정
 
 ### 3. 사용 언어
           * C++, OpenCV
